Other Conferences with SoK Papers

Since IEEE Security and Privacy started the SoK papers in 2010, several other venues have also include SoK papers.

ACM Workshop on Artificial Intelligence and Security (AISec) (started in 2012, continuing)
International Workshop on Genome Privacy and Security (GenoPri) (since 2014)
Hardware and Architectural Support for Security and Privacy (HASP) (since 2015)
IEEE European Symposium on Security and Privacy (EuroOakland) (since first edition in 2016)
USENIX Workshop on Offensive Technologies (WOOT) (since 2016)
Privacy Enhancing Technologies Symposium (PETS) (since 2016)
Attacks and Solutions in Hardware Security (ASHES) (started in 2017)
Reversing and Offensive-oriented Trends Symposium (ROOTS) (started in 2017)
Principles of Security and Trust (started in 2017)
Understanding and Enhancing On-line Privacy (2016)
IACR Fast Software Encryption (since 2017)
Security Standardization Research (since 2017)
RSA Cryptographers’ Track (since 2017)
ACM ASIA Public-Key Cryptography Workshop (since 2018)
IEEE Computer Security Foundations Symposium (since 2019)
ACM Advances in Financial Technologies (since first conference in 2019)
ACM Asia Conference on Computer and Communications Security (AsiaCCS) (since 2019)
Symposium on Usable Privacy and Security (SOUPS) (started in 2019)
Workshop on CPS & IoT Security and Privacy (since 2020)
International Conference on Availability, Reliability and Security (ARES) (since 2020)
ACM Cyber-Physical System Security Workshop (since 2021)
IEEE International Conference on Blockchain and Cryptocurrency (since 2022)
Journal of Systems Research (JSys) (since 2022)
IEEE Conference on Secure and Trustworthy Machine Learning (since first conference in 2023)



Please let me know if you know of any others that are not listed here.
